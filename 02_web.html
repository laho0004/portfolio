GRUNDLÆGGENDE WEB

I forløbet grundlæggende web har jeg lært at anvende de grundlæggende redskaber som multimediedesign studerende, som udgør en del af grundlaget for min uddannelse på MMD.  Jeg har lært grundlæggende begreber inden for design af digitale brugergrænseflader, digital kommunikation og responsivt webdesign.

Jeg har lært at oprette websider i HTML og CSS samt fået erfaring med grafik og billedbehandling og opsætning af tekst og billeder i Figma.

COMPUTERSITE

I opgaven til computersite skulle vi anvende vores viden om design konventioner, gestaltlove og wireframes. Der var fokus på "mobile first"-design med vægt på brugervenlighed og responsivt design. Vi arbejdede med CSS grid, flex og media queries.
